# NeoMutt Themes

Colour themes for [NeoMutt](https://github.com/neomutt/neomutt), the
terminal-based email client.

> **📖 Browse the full catalogue at [themes.neomutt.org](https://themes.neomutt.org/)**

## About

This repo contains over 460 colour themes for NeoMutt in two formats:

- **TrueColor** (`truecolor/`) — uses 24-bit `#RRGGBB` colours (requires
  a TrueColor-capable terminal)
- **256-colour palette** (`palette/`) — mapped to the standard 256-colour
  terminal palette

Each `.rc` file can be `source`d directly from your `neomuttrc`.

The `docs/` directory holds the source for the
[themes.neomutt.org](https://themes.neomutt.org/) documentation site, built
with [Sphinx](https://www.sphinx-doc.org/) and hosted on
[Read the Docs](https://readthedocs.org/).
